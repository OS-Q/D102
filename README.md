### [设备描述](https://github.com/OS-Q/D92/wiki) 

USB接口转换设备，采用可编程转换芯片，实现USB设备和IO设备通信间的转换，包括USB转串口等外设。

采用价格低廉的USB单片机实现USB和串口的转换。本工程用于相应协议和逻辑控制，保证输入输出接口一致。
